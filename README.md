# rock_paper_scissors
Game Rock-Paper-Scissors

## Описание
Камень, ножницы, бумага - популярная игра на руках,
часто использующаяся как способ жеребьёвки для определения
очерёдности хода или ведущего в других играх.

Победитель определяется по следующим правилам:

Бумага побеждает камень («бумага обёртывает камень»).
Камень побеждает ножницы («камень затупляет ножницы»).
Ножницы побеждают бумагу («ножницы разрезают бумагу»).
Если игроки показали одинаковый знак, то засчитывается ничья и игра переигрывается.

## Как играть
После запуска приложения выберите один из предложенных вариантов:
-камень
-ножницы
-бумага
В случае победы, счетчик Ваших побед увеличится на 1, в случае поражения - счетчик побед компьютера увеличится на 1.
Для выхода из игры напишите "выход".
Результаты игры сохранятся в файл "score.txt"

## Функциональность
Игра камень-ножницы-бумага включает в себя:
- функцию выбора хода компьютера;
- функцию получения выбора пользователя;
- функцию определения победителя;
- проверку корректности ввода пользователя;
- возможность выхода из игры.
- счетчики побед игрока и компьютера;
- метод обновления счета;
- метод отображения текущего счета;
- сохранение результатов игры в файл.

## Требования
Необходимо, чтобы все файлы приложения находились в одной директории.
Запуск приложения выполняется через файл game.py
