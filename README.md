# rock_paper_scissors
Game Rock-Paper-Scissors

## Описание
Камень, ножницы, бумага - популярная игра на руках,
часто использующаяся как способ жеребьёвки для определения
очерёдности хода или ведущего в других играх.

Победитель определяется по следующим правилам:

Бумага побеждает камень («бумага обёртывает камень»).
Камень побеждает ножницы («камень затупляет ножницы»).
Ножницы побеждают бумагу («ножницы разрезают бумагу»).
Если игроки показали одинаковый знак, то засчитывается ничья и игра переигрывается.
